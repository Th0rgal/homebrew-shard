# Homebrew Tap for Shard

This is the official [Homebrew](https://brew.sh) tap for [Shard](https://shard.thomas.md), a minimal, content-addressed Minecraft launcher.

## Installation

### CLI

```bash
brew tap th0rgal/shard
brew install shard
```

### Desktop App (Shard Launcher)

```bash
brew tap th0rgal/shard
brew install --cask shard-launcher
```

## Updating

```bash
brew update
brew upgrade shard
# or
brew upgrade --cask shard-launcher
```

## Uninstalling

```bash
brew uninstall shard
# or
brew uninstall --cask shard-launcher

# Optionally remove the tap
brew untap th0rgal/shard
```
